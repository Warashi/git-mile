# TUI バックログ

## 優先度 A

- `git-mile-tui` クレートのひな型を生成し、`cargo run` で起動できる最小構成を準備する。
- MCP セッションマネージャーを実装し、`initialize` / `shutdown` のラウンドトリップを確認する統合テストを追加する。
- list ビューの基本レイアウト（リスト + 詳細ペイン）を描画し、矢印キーで選択行が変わるようにする。

## 優先度 B

- list のフィルタ UI を実装し、プリセットの保存・読み込みを行う設定ファイル (`tui.toml`) を扱う。
- show ビューでコメントをタブ切り替え表示し、`PageUp` / `PageDown` でスクロールを制御する。
- エラーダイアログと再試行フローを UI に実装する。

## 優先度 C

- マルチセッション（既存 MCP サーバーへ接続するモード）のサポート。
- レイアウトテーマの切り替え（ライト/ダーク）とキーバインドカスタマイズ。
- Telemetry/metrics を `prometheus` 互換フォーマットで出力し、後続で可視化できるようにする。
